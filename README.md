# Telegram-бот для саммари чата — пошаговая инструкция

Инструкция рассчитана на полного новичка. Идём строго по порядку, ничего не пропускаем.

## Шаг 1. Установить Python

1. Зайти на https://www.python.org/downloads/
2. Скачать последнюю версию Python (3.11 или новее) для вашей ОС (Windows/Mac).
3. При установке на Windows **обязательно** поставить галочку "Add Python to PATH" — это внизу окна установщика.
4. Проверить, что всё встало: открыть терминал (на Windows — "Командная строка" или "PowerShell", на Mac — "Terminal") и ввести:
   ```
   python --version
   ```
   Должна показаться версия питона.

## Шаг 2. Установить редактор кода

Рекомендую **Visual Studio Code**:
1. Скачать с https://code.visualstudio.com/
2. Установить как обычную программу.
3. При первом запуске поставить расширение "Python" (иконка расширений слева, значок из квадратиков → в поиске ввести "Python" → Install).

## Шаг 3. Создать Telegram-бота

1. Открыть Telegram, найти бота **@BotFather** (это официальный бот Telegram для создания ботов).
2. Написать ему команду `/newbot`.
3. Придумать имя бота (то, что видят люди) и username (должен заканчиваться на "bot", например `my_summary_bot`).
4. BotFather пришлёт токен вида `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` — это ваш `BOT_TOKEN`, сохраните его, он понадобится дальше.
5. Важно: напишите BotFather команду `/setprivacy`, выберите вашего бота, затем **Disable**. Это разрешит боту видеть все сообщения в группе, а не только команды. Без этого шага бот не сможет собирать историю.

## Шаг 4. Получить ключ Anthropic API

1. Зайти на https://console.anthropic.com/
2. Зарегистрироваться / войти.
3. В разделе "API Keys" создать новый ключ.
4. Пополнить баланс (используется по факту, суммаризация — дешёвая операция, копейки за сводку).
5. Сохранить ключ — это ваш `ANTHROPIC_API_KEY`.

## Шаг 5. Скачать файлы проекта

Скачайте все файлы из этого набора (`bot.py`, `requirements.txt`, `.env.example`) в отдельную папку на компьютере, например `tg_summary_bot`.

## Шаг 6. Открыть проект в VS Code

1. Открыть VS Code.
2. File → Open Folder → выбрать папку `tg_summary_bot`.
3. Открыть встроенный терминал: меню Terminal → New Terminal.

## Шаг 7. Установить библиотеки

В терминале VS Code (он уже открыт в папке проекта) выполнить:
```
pip install -r requirements.txt
```
Это установит все нужные библиотеки: aiogram (работа с Telegram), anthropic (работа с Claude), apscheduler (планировщик задач по времени), python-dotenv (чтение настроек).

Если команда `pip` не находится — попробуйте `pip3` или `python -m pip install -r requirements.txt`.

## Шаг 8. Настроить токены

1. Переименовать файл `.env.example` в `.env` (просто убрать `.example` из названия).
2. Открыть `.env` в VS Code и вписать туда реальные значения:
   ```
   BOT_TOKEN=ваш_токен_от_botfather
   ANTHROPIC_API_KEY=ваш_ключ_anthropic
   ```
3. Сохранить файл (Ctrl+S / Cmd+S).

**Важно:** никогда не публикуйте `.env` файл и не отправляйте его никому — это ваши секретные ключи.

## Шаг 9. Добавить бота в чат/канал

1. Открыть свой чат или группу в Telegram.
2. Добавить туда бота как участника (через "Добавить участника" и поиск по username бота).
3. Если это группа — по возможности сделать бота администратором (так надёжнее видит все сообщения).

## Шаг 10. Узнать ID чата (chat_id)

Это нужно, чтобы бот знал, куда автоматически публиковать ежедневную сводку.

Способ попроще:
1. Добавьте в свой чат бота **@userinfobot** — он покажет ID чата.
2. Скопируйте это число (для групп оно обычно отрицательное, например `-1001234567890`).
3. Откройте `bot.py`, найдите строку:
   ```python
   TARGET_CHAT_ID = None
   ```
   и замените `None` на ваш ID, например:
   ```python
   TARGET_CHAT_ID = -1001234567890
   ```

Без этого шага бот всё равно будет работать, но только по ручной команде `/summary`, без автоматической ежедневной рассылки.

## Шаг 11. Запустить бота

В терминале VS Code выполнить:
```
python bot.py
```
Если всё настроено верно, в терминале появится строка "Бот запущен". Не закрывайте терминал — пока он открыт, бот работает.

## Шаг 12. Проверить работу

1. Напишите что-нибудь в чат, где сидит бот.
2. Напишите боту команду `/summary` — он должен собрать сообщения за сутки и прислать сводку.
3. Если настроен `TARGET_CHAT_ID`, каждый день в 22:00 (по времени GMT+3, вы можете поменять `SUMMARY_HOUR` в bot.py) бот будет сам присылать сводку.

## Шаг 13. Чтобы бот работал круглосуточно

Пока вы запускаете `python bot.py` на своём компьютере — бот работает только пока открыт терминал и включён компьютер. Чтобы бот работал 24/7 без вашего участия, нужно разместить его на сервере. Варианты для новичка:
- **Railway.app** или **Render.com** — можно бесплатно или почти бесплатно развернуть такой скрипт, залив код на GitHub и подключив репозиторий.
- Обычный VPS (например, Timeweb, Selectel) — чуть сложнее в настройке, но даёт больше контроля.

Это отдельный этап — если дойдёте до него, могу расписать подробно и его.

## Частые проблемы

- **Бот не видит сообщения в группе** → проверьте шаг 3 (Disable Privacy Mode) и что бот состоит в чате.
- **Ошибка с токеном** → проверьте, что `.env` называется именно `.env` (без `.example`) и лежит в той же папке, что `bot.py`.
- **ModuleNotFoundError** → значит библиотеки не установились, повторите Шаг 7.
